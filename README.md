# Business Analytics & KPI Dashboard

An interactive, data-driven web application built to bridge the gap between front-end engineering and corporate Business Intelligence (BI). This tool processes raw financial and traffic data to render clean, operational charts for stakeholder decision-making.

## Key Features
* **Interactive Data Visualization:** Integrates modern charting frameworks to display real-time trends, revenue goals, and overhead metrics.
* **Metric Aggregation:** Programmatically calculates high-level KPIs like Monthly Recurring Revenue (MRR), Customer Acquisition Cost (CAC), and operational margins from structured data arrays.
* **Responsive Layouts:** Designed using modern CSS Grid and Flexbox layers to ensure data is completely readable across mobile devices and desktop monitors.

## Tech Stack & Concepts
* **Front-End Design:** Semantic HTML5 and modular CSS3 architecture.
* **Data Processing:** Vanilla JavaScript logic arrays for filtering, sorting, and cleaning raw datasets.
